这个文件是在 `cli/main.py` 里被调用的。具体位置在 `get_user_selections()` 函数中：

```python
with open(Path(__file__).parent / "static" / "welcome.txt", "r") as f:
    welcome_ascii = f.read()
```

也就是说，CLI 启动后在收集用户参数之前，会先读取 `cli/static/welcome.txt` 的内容，把它当作欢迎界面的 ASCII 艺术字，再拼到欢迎面板里显示。([GitHub][1])

### 它的用途

这个 `welcome.txt` 不是业务逻辑文件，也不是配置文件，而是一个**静态欢迎文本资源**，主要用于美化 CLI 启动画面。它被读出来后会和下面这些内容一起组成欢迎界面：

* `TradingAgents: Multi-Agents LLM Financial Trading Framework - CLI`
* Workflow Steps
* 后续的交互提示。([GitHub][1])

### 调用链

可以理解成：

```text
cli/main.py
  → get_user_selections()
    → 读取 cli/static/welcome.txt
    → 拼接欢迎内容
    → 在终端显示欢迎界面
```

所以，`welcome.txt` 的调用位置就是 `cli/main.py` 里的 `get_user_selections()`。([GitHub][1])
